# CountMe

1-Download and install XAMPP for hosting locally your website 
2-Move the website to -> C:\xampp\htdocs
3-Run XAMPP Apache and XAMPP MySQL from XAMPP terminal
4-Go to 'localhost/phpmyadmin' on your browser
5-Go home on your local-server (phpMyAdmin)
6-Click on Import button from the page header, select bara_natanael_create.sql -> path 'C:\xampp\htdocs\CountMe\db\physical_model' and Execute
7-Copy the link for your website: 'localhost/CountMe'
8-ENJOY!
